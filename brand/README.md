# Vliegseintje — brand-kit

Alle logo's en merk-assets om overal te uploaden. Hieronder welk bestand je waarvoor pakt.

## Logo's (vector — gebruik deze waar het kan)
- **vliegseintje-mark.svg** — alleen het icoon (het vliegtuigje + amber ping). Schaalt oneindig scherp.
- **vliegseintje-logo.svg** — icoon + "Vliegseintje" ernaast, **donkere** tekst (voor lichte achtergrond).
- **vliegseintje-logo-white.svg** — zelfde lockup met **witte** tekst (voor donkere/blauwe achtergrond).
- **favicon.svg** — vereenvoudigde versie voor browser-tabjes.

## PNG's (als een plek geen SVG accepteert)
- **png/mark/** — alleen het icoon, transparante achtergrond, 16 t/m 1024 px.
- **png/logo/** — icoon + naam, transparant, 600/1200/2400 px breed. `…-white-…` = witte tekst voor donkere achtergrond.
- **png/app-icon/vliegseintje-app-icon-1024.png** — vierkant, vol-blauw (géén transparantie). Voor app-stores, profielfoto's en het **Lemon Squeezy store-logo**.
- **png/favicon/** — favicons (16/32/48), apple-touch-icon (180), PWA-iconen (192/512) en `favicon.ico`.

## Social
- **social/og-image.jpg** — 1200×630 deel-beeld voor link-previews (WhatsApp / X / LinkedIn).

## Huisstijl
**Kleuren** — Primair blauw `#2563EB` · Accent amber `#FFB703` · Donker/tekst `#102A43` · Licht/achtergrond `#F5F8FC`
**Lettertypes** — Koppen & logo: **Poppins** (Bold/700) · Lopende tekst: **Inter**

## Snelgids: waar wat uploaden
| Plek | Bestand |
|---|---|
| Lemon Squeezy store-logo (Settings → Design) | `png/app-icon/vliegseintje-app-icon-1024.png` |
| Social-media profielfoto / avatar | `png/app-icon/vliegseintje-app-icon-1024.png` |
| Social-media omslag op donkere achtergrond | `png/logo/vliegseintje-logo-white-2400.png` |
| E-mailhandtekening | `png/logo/vliegseintje-logo-600.png` |
| Website-favicon | `favicon.svg` + `png/favicon/` |
| Link-preview / OG-beeld | `social/og-image.jpg` |

> De brand-kit-mark is bewust nét iets schoner dan de mini-versie in de website-header
> (zonder de faint "ping-ring"): die ring blendt op groot formaat grijs. Wil je de header
> identiek maken aan deze kit, dan pas ik dat in één tik aan.
